## 2. Trignometry
```python
a = np.array([0,30,45,60,90,135,180,270,360])
```

Degree to Radian
```python
np.deg2rad(a)
```

Radian to Degree
```python
r = np.deg2rad(a)
np.rad2deg(r)
```

## 3. Power, Log, Exponent
```python
x = np.array([1,3,4,6,9,15,18,27])
```

Power
```python
np.power(x, 2)  # power of 2
```

Expoenntial
```python
np.exp(x)  # e^x
```

Log
```python
np.log(x)  # ln(x)
np.log10(x)  # log(x) with base 10

```
